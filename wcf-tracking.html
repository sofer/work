Notes

To add tracking for files, simply place tracking.js inside the head tags at any point after loading jquery as follows:

<script src="<path-to-js-files>/tracking.js" type="text/javascript"></script>

Remove the second line of tracking.js and uncomment the third line. Then load tracking.js and this page (with the path amended as above and the GA account and domain amended) onto the womenandchildrenfirst.org.uk domain, in order for google to start tracking file clicks.

The list of file paths is on the first line of the tracking.js file and can be amended. Use lower case only in the definitions (although the files themselves can be upper or mixed case, too). pdf, ppt, doc, docx, pptx, xls and xlsx are all currently recognised.

Any links outside the current host will be ignored for tracking purposes.

It looks like there is a bit of namespace conflict between jQuery and mootools, so we are using $jpcc in place of $.
